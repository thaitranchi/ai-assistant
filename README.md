# AI Support Assistant API

## Overview
This project is a simple AI-powered backend service that integrates LLM APIs to automate responses, summarization, and content generation.

## Features
- Chat API (Q&A)
- Text Summarization
- Content Generation
- RESTful API with FastAPI

## Tech Stack
- Python (FastAPI)
- OpenAI API
- REST API

## Run locally
```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
